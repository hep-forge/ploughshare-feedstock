Feedstock for [ploughshare](https://ploughshare.web.cern.ch) — part of [hep-forge](https://anaconda.org/hep-forge).
Builds linux-amd64 + linux-arm64 in one matrix workflow and uploads to the
[hep-forge](https://anaconda.org/hep-forge) Anaconda channel.

Ploughshare for all your interpolation grid needs

## Architectures

| Architecture | Latest published |
|--------------|------------------|
| linux-amd64 (`linux-64`) | ✅ `0.0.20` |
| linux-arm64 (`linux-aarch64`) | ✅ `0.0.20` |

_As of the last feedstock render; the badges above are live._


## Install

```bash
conda install -c hep-forge -c conda-forge ploughshare
```
